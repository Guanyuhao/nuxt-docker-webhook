# 项目概要

## 本地开发时可能用到的命令
```bash
# 安装依赖
npm i
# 启动项目
npm run dev
# 带覆盖率的测试
npm run test
# 全量测试,适合开发时使用
npm run ltest --watch
# 单独跑一个测试文件
npm run ltest test/file-to-test.js --watch
# 查看开发文档
npm i gitbook-cli -g
gitbook serve
open http://localhost:4000
```
